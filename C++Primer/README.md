# 代码规范：

* 定义类后要加分号
* 直接初始化代替拷贝初始化
* 迭代器使用!=
* 常量指针用作形参可避免拷贝
* 尽量使用标准库而非数组和指针，尽量使用string而非C风格数组字符串
